Claude Codeとは？

ターミナルで動くAIコーディングアシスタント

Claude Codeは、Anthropic社が開発したAIコーディングツールです。 ターミナル上で自然言語の指示を出すだけで、コードの生成・修正・リファクタリング・デバッグまでAIが一緒にやってくれます。

「プログラミングは難しそう…」と思っていた方も、Claude Codeを使えば 日本語で指示するだけでコードが書ける時代に。 この勉強会で、その威力を体感してみませんか？

// Claude Codeに指示を出すだけ

あなた > ユーザー登録フォームを
        バリデーション付きで作って

Claude > 了解！フォームを作成します。
  const validateForm = (data) => {
    if (!data.email.includes('@'))
      return { error: 'メール形式不正' }
    ...
  }
